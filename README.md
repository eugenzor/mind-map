# Mind Map: Навчитися вчитися (Барбара Оклі)

Цей проєкт містить інтелектуальну карту (mind map) за мотивами бестселера Барбари Оклі **"Навчитися вчитися"**. Карта
структурує ключові концепції книги, такі як сфокусоване та розсіяне мислення, боротьба з прокрастинацією та техніки
запам'ятовування.

## Про формат OPML

Основним форматом даних у цьому проєкті є **OPML** (*Outline Processor Markup Language*). Це стандартний формат на
основі XML, який використовується для зберігання ієрархічних структур.

### Переваги формату OPML:

* **Універсальність:** Його розуміють майже всі програми для створення інтелектуальних карт та планувальники завдань.
* **Легкість:** Файл займає мінімум місця, оскільки містить лише текст та структуру.
* **Відсутність прив'язки до софту:** Ви можете почати роботу в одному додатку, а продовжити в іншому, не втрачаючи
  логіки зв’язків.
* **Текстова основа:** Файл можна відкрити навіть звичайним блокнотом, щоб переглянути ієрархію.

## Інструменти для роботи з OPML

Ви можете відкрити, редагувати або візуалізувати файл проєкту за допомогою таких інструментів:

1. **[Xmind.ai / Xmind.app](https://xmind.app)** — один із найпопулярніших інструментів. Має спеціальний
   онлайн-конвертер [OPML to Mind Map](https://www.google.com/search?q=https://xmind.app/tools/opml-to-mind-map/).
2. **MindMeister** — потужний веб-сервіс для спільної роботи, що підтримує імпорт OPML.
3. **MindNode** — ідеальний вибір для користувачів macOS та iOS.
4. **FreeMind / FreePlane** — безкоштовні інструменти з відкритим кодом.
5. **Obsidian** — через плагіни (наприклад, Mind Map) дозволяє перетворювати структуровані списки в карти.

## Як використовувати цей проєкт

1. Завантажте файл `.opml` з цього репозиторію.
2. Перейдіть на сайт [xmind.app](https://xmind.app) або відкрийте будь-який інший сумісний додаток.
3. Оберіть функцію **"Import"** та виберіть завантажений файл.
4. Налаштуйте візуальний стиль на свій смак\!
